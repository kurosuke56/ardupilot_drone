# XIAO ESP32S3 4モーターテスト

XIAO ESP32-S3 を使用して、4個のブラシモーターを順番にPWM制御で駆動するテストプログラムです。  
モーターの動作確認、配線チェック、モータードライバの基本動作検証を目的としています。

説明記事（note）
https://note.com/kurosuke_9481/n/ndace4f24b80d

---

## 概要

このプログラムでは、4つのモーターを1つずつ順番にテストします。

各モーターに対して以下の動作を行います：

- PWMデューティ比 0 → 255 へ徐々に増加（加速）
- 短時間フル回転
- PWMデューティ比 255 → 0 へ徐々に減少（減速）
- 停止

テストは M1 → M2 → M3 → M4 の順に繰り返されます。

---

## 使用環境

- マイコン: Seeed Studio XIAO ESP32-S3
- 開発環境: Arduino IDE
- PWM周波数: 20kHz
- PWM分解能: 8bit（0〜255）

---

## ピン配置

### モーター出力

| モーター | GPIO |
|----------|------|
| M1       | GPIO1 |
| M2       | GPIO2 |
| M3       | GPIO7 |
| M4       | GPIO8 |

### 内蔵LED

| 機能 | GPIO |
|------|------|
| LED  | GPIO21 |

※ 内蔵LEDは LOW 出力で点灯します。

---

## 配線例

```text
XIAO ESP32-S3
 ├─ GPIO1 → Motor Driver IN1
 ├─ GPIO2 → Motor Driver IN2
 ├─ GPIO7 → Motor Driver IN3
 └─ GPIO8 → Motor Driver IN4
```

DRV8833などのHブリッジモータードライバの入力ピンに接続してください。

## 動作フロー
1. シリアル通信開始（115200bps）
2. 内蔵LEDをON（LOW）
3. 全モーター停止状態に初期化
4. 各モーターを順番にテスト

## 用途
- モーター単体の動作確認
- モータードライバ（DRV8833など）の配線チェック
- PWM制御の基本動作確認
- 自作ドローンの初期検証

## 注意事項
- モーター電源はUSBだけでは不足する場合があります
- プロペラを付けた状態でのテストは危険です
- 必ずモーター固定状態で実施してください
